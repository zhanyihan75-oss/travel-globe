# Project Context

## Why this project exists

Travel Globe is **not** a traditional personal portfolio.

It is a product case study that demonstrates: - Product Thinking - UX
Design - UI Design - Front-end Engineering - AI-assisted Development

The project revolves around one core concept:

> **Destination**

Everything begins with the 3D globe and ends at the Destination Page.

## Core Philosophy

-   The map is the product.
-   Every feature should strengthen the Destination Page.
-   Simplicity is more valuable than feature count.
-   Build an excellent MVP before expanding.

## Success Criteria

A visitor should understand the product within 30 seconds: 1. Explore
the globe. 2. Click a destination. 3. View personal travel memories. 4.
Get inspiration for future travel.

Future features (Community / AI) should always support this journey.
